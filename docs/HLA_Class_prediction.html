Evaluate some algorithms to this classification problem
First separate the dataset into train and test sets
array =  working_dataset.values
X = array[:, 0:4] # all independent values
Y = array[:, 4] # dependent or response value
validation_size = 0.20 # 20 % for this instance
seed = 123
X_train, X_validation, Y_train, Y_validation = model_selection.train_test_split(X, Y, test_size = validation_size,
random_state = seed)
# numpy array objects are created by train_test_split function
# Set seed again and assign 'accuracy' to scoring variable because KFold function below will
# give accuracy values that we need to assign to scoring parameterself.
seed = 123
scoring = 'accuracy'
Build the models together and compare accuracy
# We will build 7 different models for this classification problem . They are :
# Logistic Regression (LR)
# Linear Discriminant Analysis (LDA)
# K-Nearest Neighbors (KNN).
# Classification and Regression Trees (CART).
# Gaussian Naive Bayes (NB).
# Support Vector Machines (SVM).
# RandomForestClassifier (RF)

models = []
models.append(('LR', LogisticRegression()))
models.append(('LDR', LinearDiscriminantAnalysis()))
models.append(('KNN', KNeighborsClassifier()))
models.append(('CART', DecisionTreeClassifier()))
models.append(('NB', GaussianNB()))
models.append(('SVM', SVC()))
models.append(('RF', RandomForestClassifier()))
Build the models sequentially and evaluate their accuracy with the model_selection function.Remember validation set will not be used here . We will use 10 fold cross validation method to generate cv score and scoring values for model_selection.cross_val_score function below.
names = []
results = []
for name, model in models:
	kfold = model_selection.KFold(n_splits=10, random_state=seed)
	cv_results = model_selection.cross_val_score(model, X_train, Y_train, cv=kfold, scoring=scoring)
	results.append(cv_results)
	names.append(name)
	msg = "%s: %f (%f)" % (name, cv_results.mean(), cv_results.std())
	print(msg)
